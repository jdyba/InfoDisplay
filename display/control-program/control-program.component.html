App information

It is used in coach to inform passengers standing on bus stop

This app was created to illustrate a real, physical device. I created that display on my own using some electrical parts among others: led matrix, lcd screen, membrane keyboard controlled by arduino board programmed in C language. The idea of programming it is that the app should reflect the device in one hundred percent. This app can help people use a real device. In this app you can:

  • choose a subtitle from memory
  • add a new subtitle using a specific keyboard(like in old phones)
  • remove a subtitle from memory
  • display a information on the matrix
